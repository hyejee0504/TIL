### - VSC 단축어, 편리한 기능 정리
        1. ctrl + N : 새 파일 열기
        2. ctrl + S : 다른이름으로 저장
        3. ! + Tab : 기본적인 문법 자동완성
        4. ctrl + alt + 방향키 : 동시수정 가능
        5. html.json 내에서 기본 defualt값 변경 가능 
         - htmlko 자동완성으로 실행
        6. div{hello}*10 -> div태그 내에 hello라는 
        내용으로 10개 복사
